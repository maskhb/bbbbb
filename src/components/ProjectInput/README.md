# 项目选择输入框

## API

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| data | 项目数据 | Array[{}] | - |
| value       | 默认值     | Array[省, 市, 区]  | -    |
| convertToOptions | 项目数据转化ant 数据的方法    | function  | -    |
| style     | 式样,默认 width: 200px     | Object  | -    |
